## 📌 Project Overview
This repository contains the three iterative stages of my Sketch 3 project for Term 2. 
It explores embodied interaction and frictionless digital aesthetics using hand-tracking.

### Folders Included:
- **Interaction1**: The initial rigid line connection (Feature test).
- **Interaction2**: Complex gesture mapping (Caused performance drops).
- **Final**: The optimized, physics-based fluid ribbon with history lag (Final Version).

## 🎮 Instructions (Operation Manual)
To run the **Final** version:
1. Open the `index.html` file inside the `Final` folder using a local server (like VS Code Live Server) or simply double-click it.
2. **Allow Camera Access** when the browser prompts you.
3. Stand 1-2 meters away from the webcam.
4. Raise both hands. The system will track your **Index Fingers** (keypoints 8).
5. Move your hands slowly to draw and manipulate the digital silk. The system uses a physics delay to create a smooth, glowing ribbon effect.
